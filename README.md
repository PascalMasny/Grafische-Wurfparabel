# Installation
1. [Python 3](https://www.python.org/)
2. [Gnuplot](http://www.gnuplot.info/)
```
git clone https://github.com/PasiXD189/Wurfparabel.git
cd Wurfparabel
sudo ./install.sh
```

# Nutzung Linux
```
./start.sh ##öffnen
##Ergebniss: wurf.png 
```

# Nurzung in Windows
```
wurf.py ##öffnen und Werte eingeben
wurf.plt ##öffnen => wurf.png wird generiert
##Ergebniss: wurf.png
```
